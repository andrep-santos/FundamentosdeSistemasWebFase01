# Petshop Amigo Fiel - Fase 1

## Sobre o Projeto
Este projeto consiste no desenvolvimento da estrutura front-end de um sistema web para o Petshop Amigo Fiel. O objetivo é apresentar produtos e serviços da loja de forma clara e acessível.

Nesta Fase 1, o foco foi a estruturação semântica utilizando HTML5 puro, sem o uso de CSS frameworks ou JavaScript, garantindo uma base sólida para as próximas etapas.

## Funcionalidades
* **Navegação:** Menu superior e rodapé com links de acesso rápido para as seções (Home, Produtos, Serviços, Contato).
* **Catálogo de Produtos:** Exibição de produtos divididos por categorias (Acessórios, Rações, Higiene), contendo imagem, descrição e valor.
* **Lista de Serviços:** Apresentação dos serviços de Banho e Tosa, com diferenciação de preços para serviços com ou sem tele-busca.
* **Contato:** Informações de SAC e localização no rodapé.

## Tecnologias Utilizadas
* HTML5 (Semântico)
